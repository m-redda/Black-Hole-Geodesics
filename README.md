# Black Hole Geodesics

A Python project for the symbolic derivation, numerical simulation, and visualisation of particle motion in curved spacetime.

This repository recreates and extends the work from my MPhys dissertation, **"Black Hole Spacetimes: Testing the Limits of General Relativity"**, by implementing the underlying mathematics and simulations using modern Python scientific libraries.

Rather than simply translating Mathematica notebooks, this project rebuilds each stage of the analysis—from the metric tensor to orbit simulations—in a modular, reproducible, and open-source format.

---

## Project Goals

This project aims to

- Symbolically derive the equations governing particle motion in black hole spacetimes using SymPy.
- Numerically solve the resulting geodesic equations using SciPy.
- Produce publication-quality figures with Matplotlib.
- Reproduce the results from my MPhys dissertation.
- Create an educational and reusable computational relativity toolkit.
